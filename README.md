# IPL Analysis Dashboard using Power BI

## Project Overview

This project is an interactive Power BI dashboard built using IPL match data. It provides insights into team performance, player statistics, match outcomes, and season-wise trends through dynamic visualizations.

---

## Dashboard Features

- Interactive slicers and filters
- Team-wise analysis
- Season-wise analysis
- Top run scorers
- Top wicket takers
- Toss decision analysis
- Match result analysis
- KPI Cards

---

## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel

## Skills Demonstrated

- Data Cleaning

- DAX Calculations
- Dashboard Design
