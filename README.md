# 🏏 IPL Analysis Dashboard | Power BI

An interactive Power BI dashboard designed to analyze Indian Premier League (IPL) data and uncover insights into team performance, player statistics, match outcomes, and season-wise trends. This project demonstrates skills in data cleaning, data modeling, DAX, and dashboard design.

---

## 📌 Project Overview

The objective of this project is to transform raw IPL data into an interactive business intelligence dashboard that enables users to explore historical IPL trends through dynamic visualizations and filters.

The dashboard allows users to analyze:

- Team performance across seasons
- Player statistics
- Match-winning patterns
- Toss decisions and their impact
- Venue-wise match distribution
- Season-wise IPL trends

---

## 🎯 Objectives

- Clean and transform IPL datasets using Power Query.
- Build an interactive dashboard using Power BI.
- Create meaningful KPIs using DAX.
- Enable dynamic filtering for better analysis.
- Present insights through clear and intuitive visualizations.
